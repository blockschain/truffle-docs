Ganache UI 是支持 Etereum 和 Corda 技术的桌面应用程序。
此外，Ethereum 版本的 GaNache 可用作命令行工具: [ganache-cli](https://github.com/trufflesuite/ganache-cli) (以前称为 TestRPC).
所有版本的 GaNache 都适用于 Windows，Mac 和 Linux。
